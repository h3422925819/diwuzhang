# 第5章主题式数据可视化网站

这是一个基于第5章内容重新设计的主题式数据可视化网站，采用以主题为基础的子图布局，提供更加直观和专业的数据展示体验。

## 📁 项目结构

```
第5章主题式数据可视化网站/
├── index.html                # 主页面
├── css/
│   └── style.css            # 主题式布局样式
├── js/
│   └── app.js               # 主题化JavaScript逻辑
├── data/
│   ├── theme_data.json      # 主题数据文件
│   ├── sample_data.json     # 通用示例数据
│   ├── sales_data.csv       # CSV销售数据
│   └── region_data.csv      # CSV区域数据
└── README.md                # 说明文档
```

## 🎯 设计特色

### 主题式布局
- **5个主题模块**: 销售分析、区域分析、客户分析、趋势分析、绩效分析
- **20个子图**: 每个主题包含4个专业子图
- **响应式网格**: 自适应不同屏幕尺寸的网格布局
- **主题切换**: 平滑的主题切换动画

### 子图类型
1. **销售分析主题**
   - 5.1.1 月度销售趋势（折线图/面积图）
   - 5.1.2 产品销售分布（饼图/柱状图）
   - 5.1.3 销售目标达成（柱状图）
   - 5.1.4 销售渠道分析（环形图）

2. **区域分析主题**
   - 5.2.1 区域销售分布（横向柱状图）
   - 5.2.2 区域增长趋势（多线图）
   - 5.2.3 市场份额分析（漏斗图）
   - 5.2.4 区域活跃度热力图

3. **客户分析主题**
   - 5.3.1 客户结构分析（饼图）
   - 5.3.2 客户满意度评估（柱状图）
   - 5.3.3 客户生命周期（混合图）
   - 5.3.4 客户行为模式（雷达图）

4. **趋势分析主题**
   - 5.4.1 长期发展趋势（多线图）
   - 5.4.2 季节性模式（多线对比）
   - 5.4.3 业务预测（预测区间图）
   - 5.4.4 指标相关性（相关性热力图）

5. **绩效分析主题**
   - 5.5.1 KPI仪表盘（对比柱状图）
   - 5.5.2 绩效排名（横向柱状图）
   - 5.5.3 效率分析（混合图）
   - 5.5.4 改进建议（散点图）

## 🔧 自定义配置

### 添加新主题
```javascript
// 在app.js中添加新主题初始化方法
initNewThemeCharts(data) {
    // 初始化新主题的4个子图
    this.charts['new-chart-1'] = echarts.init(document.getElementById('new-chart-1'));
    // 配置图表选项...
}
```

### 修改主题样式
```css
/* 在style.css中自定义主题颜色 */
.theme-content[data-theme="new-theme"] {
    --primary-color: #your-color;
    --secondary-color: #your-secondary-color;
}
```

### 添加新子图
1. 在HTML中添加子图容器
2. 在CSS中定义样式
3. 在JavaScript中初始化图表
4. 在数据文件中添加数据
